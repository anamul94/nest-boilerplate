## Installation

```bash
$ npm install
```

## Running the app

Create Role in DB manually

```bash
# Prepare .env
$ cp .env-example .env
# set value according to your config

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

## DB migration

```bash
# generate
$ npm run migration:generate db/migrations/<name_of_migration>

#run
$ npm run migration:run

#revert

$ npm run migration:revert

```
